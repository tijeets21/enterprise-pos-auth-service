Detailed Description

Authentication helpers and middleware for ProjectV.

Provides JWT token generation and verification, and attaches the decoded user object to the request for downstream handlers.

function export authenticate ( req,
res,
next )

Express middleware for JWT authentication.

Reads the Authorization header, verifies the JWT using the configured secret, and attaches the decoded payload to req.user. If the token is missing or invalid, responds with HTTP 401.

Parameters
{import("express").Request}req The incoming request.
{import("express").Response}res The HTTP response.
{Function}next Next middleware handler.
