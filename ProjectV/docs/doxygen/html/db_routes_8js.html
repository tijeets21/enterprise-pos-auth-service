Detailed Description

Routes for managing collections and documents in MongoDB.

Provides endpoints to list and create collections, and to perform CRUD and soft-delete operations on documents within those collections.

function coerceValue ( value)

Coerce string query values into appropriate types.

Converts "true"/"false" to booleans, "null" to null, and numeric strings to numbers where possible.
