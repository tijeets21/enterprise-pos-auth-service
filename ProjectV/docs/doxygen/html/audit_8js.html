Detailed Description

Request/response audit logging middleware.

Records details about each HTTP request and response in the actions MongoDB collection, including user, path, status, duration, IP address, and user agent.

Middleware that logs API actions to MongoDB.

Wraps the response lifecycle to capture timing and status. After the response is sent, this middleware writes an audit entry to the actions collection for debugging and compliance.

Logged fields include:

  • username, email (from req.user when available)
  • HTTP method and full path
  • params, query, body
  • statusCode and duration (ms)
  • timestamp, IP address, user agent
